No. Currently it doesn't work.

Is there a configuration file?

How do I restart the lwp server?

Can I install it over an existing LXC configuration?
Yes. Gantry is made to do this.

How can I contribute to Gantry?
Fork Gantry on Github

Can I use Apache2 WSGI mod?
